¿Qué es una base de datos relacional?

Una base de datos relacional es una forma de guardar información de manera organizada, usando tablas que están conectadas entre sí. Es similar a tener varias hojas de cálculo donde cada una tiene su propia información, pero todas están relacionadas mediante claves.

Por ejemplo, en este proyecto se tiene una tabla de estudiantes y otra de materias. En lugar de escribir el nombre del docente en cada registro, solo se coloca un identificador numérico que hace referencia al docente en su propia tabla. Esto evita la repetición de datos y hace todo más ordenado y eficiente.

Este modelo fue propuesto por Edgar F. Codd en 1970 y sigue siendo el estándar más utilizado en sistemas de información a nivel mundial, tanto en aplicaciones locales como en entornos de nube.

Ventajas de las bases de datos relacionales

  • Sin redundancia de datos

    La información no se repite entre tablas, lo que ahorra espacio de almacenamiento y facilita el mantenimiento.

  • Consultas flexibles

    Es más fácil buscar, filtrar y cruzar datos usando el lenguaje SQL, incluso entre múltiples tablas al mismo tiempo.

  • Integridad de datos

    Si se necesita modificar un dato, solo se cambia en un lugar y el cambio se refleja automáticamente en toda la base de datos gracias a las relaciones definidas.

  • Seguridad y control de acceso

    Permiten definir permisos y roles de usuario, protegiendo la información sensible de accesos no autorizados.

Gestores Utilizados

Para este proyecto se utilizaron dos tecnologías principales que trabajan en conjunto: MySQL como sistema gestor de base de datos y Hostinger como plataforma de hospedaje en la nube.

MySQL

MySQL es uno de los sistemas gestores de bases de datos relacionales más conocidos y utilizados en el mundo. Permite crear, almacenar y organizar información de forma estructurada mediante tablas relacionadas entre sí. Fue desarrollado originalmente por MySQL AB y actualmente es mantenido por Oracle Corporation.

  • Gratuito y de código abierto

    No requiere licencias de pago para su uso en proyectos educativos o de desarrollo, lo que lo hace ideal para este tipo de proyectos.

  • Compatible con phpMyAdmin

    Funciona perfectamente con phpMyAdmin, la herramienta visual utilizada para crear y administrar las tablas de forma gráfica sin necesidad de escribir comandos manualmente.

  • Amplia compatibilidad

    Es compatible con la mayoría de los lenguajes de programación web como PHP, Python y JavaScript, así como con plataformas de hospedaje como Hostinger.

  • Alto rendimiento

    Está optimizado para manejar grandes volúmenes de datos con rapidez, siendo confiable tanto en entornos locales como en la nube.

Hostinger

Hostinger es un servicio de hospedaje web que también incluye bases de datos MySQL. Está pensado principalmente para alojar páginas web junto con su base de datos, aunque también puede utilizarse únicamente para gestión de datos en la nube. Es uno de los proveedores de hospedaje más accesibles del mercado.

  • phpMyAdmin integrado

    Incluye phpMyAdmin de forma nativa, lo que facilita enormemente la importación, visualización y gestión de la base de datos directamente desde el panel de control.

  • Proceso de importación sencillo

    El procedimiento para subir la base de datos a Hostinger es idéntico al proceso local, lo que reduce la curva de aprendizaje y evita configuraciones complicadas.

  • Confiabilidad y disponibilidad

    Ofrece alta disponibilidad del servicio con copias de seguridad automáticas, lo que protege los datos ante posibles fallos o pérdidas accidentales.

  • Panel de control intuitivo

    Su interfaz de administración (hPanel) es clara y fácil de navegar, permitiendo gestionar dominios, bases de datos y archivos desde un solo lugar.
