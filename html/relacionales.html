Bases de datos relacionales en la nube

Aprende cómo se organizan los datos en tablas conectadas y cómo migrarlos a la nube.

0% completado

Concepto

¿Qué es una base de datos relacional?

Es una forma de guardar información de manera organizada usando tablas conectadas entre sí. Se puede imaginar como varias hojas de cálculo donde cada una tiene su propia información, pero todas están relacionadas.

Tablas Sin repetición Fácil de filtrar Un cambio, todo actualizado

Clave primaria

Identificador único de cada registro dentro de una tabla.

Clave externa

Conecta dos tablas creando una relación lógica entre sus datos.

Gestor

Gestor de base de datos: MySQL

MySQL es uno de los sistemas de gestión de bases de datos más utilizados en el mundo. Permite crear, guardar y organizar información en tablas de forma estructurada mediante el lenguaje SQL.

Gratuito y de código abierto Cuenta con una gran comunidad de soporte y documentación disponible.
Administración visual con phpMyAdmin Permite gestionar la base de datos sin necesidad de escribir código directamente.
Compatible con la nube Funciona con la mayoría de servicios de hospedaje en la nube disponibles.

Si se necesita modificar un dato, solo se cambia en un lugar y la actualización se refleja en toda la base de datos.

Proceso

¿Cómo se sube una base de datos a la nube?

El proceso consiste en exportar la base de datos desde el entorno local e importarla en el servicio de nube elegido. A continuación se muestra el proceso usando Hostinger como ejemplo:

1
Exportar desde el entorno local

Desde phpMyAdmin (usando XAMPP en local) se exporta la base de datos en formato .SQL. Este archivo contiene todas las instrucciones para recrear las tablas y sus datos.

2
Crear la base de datos en Hostinger

Desde el panel de control de Hostinger se crea una nueva base de datos MySQL y se accede a su phpMyAdmin integrado.

3
Importar el archivo SQL

Se selecciona la base de datos recién creada, se usa la opción Importar y se carga el archivo .SQL. El proceso es idéntico al que se usa en local.

Proveedores

Otros servicios disponibles

Hostinger es solo una opción. Existen otros proveedores reconocidos donde también se puede alojar una base de datos relacional en la nube:

A
Amazon RDS (AWS)

Soporta MySQL, PostgreSQL, Oracle y SQL Server. Ideal para proyectos escalables.

G
Google Cloud SQL

Servicio gestionado para MySQL, PostgreSQL y SQL Server en la nube de Google.

M
Azure SQL Database

Base de datos inteligente y escalable en la nube de Microsoft Azure.
