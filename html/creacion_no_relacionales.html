Creación de bases de datos no relacionales

Crear una base de datos no relacional en la nube significa construir un sistema de almacenamiento de información que, en lugar de organizar los datos en tablas fijas con filas y columnas, utiliza estructuras más flexibles (documentos, pares de clave y valor, o grafos.)

Para la creacion de este tipo de bases de datos solo se requiere crear una cuenta en el proveedor elegido, seleccionar el tipo de servicio y comenzar a trabajar. Esto permite que cualquier persona con acceso a internet pueda gestionar su base de datos desde cualquier lugar y en cualquier momento.

Existen varios metodos para crear bases de datos no relacionales:

  • Usando la interfaz web del proveedor de servicios en la nube
  • Utilizando herramientas de línea de comandos
  • Implementando APIs para la creación automática
  • Creación programática desde una aplicación
  • Creación automática al insertar datos

Naturalmente, cada método tiene sus ventajas y desventajas, y la elección del método adecuado dependerá de las necesidades específicas del proyecto. Normalmente el método mas utilizado es el de la interfaz web, ya que ofrece una experiencia de usuario más sencilla y accesible.

A continuacion se muestra un claaro ejemplo de la creación de una base de datos en el entorno de MongoDB:

  • Ingresar a MongoDB Atlas

    Acceder al sitio oficial de MongoDB Atlas e iniciar sesión con una cuenta existente o registrarse de forma gratuita.

  • Crear un proyecto

    Crear un nuevo proyecto para organizar y administrar los recursos relacionados con la base de datos.

  • Crear un cluster

    Configurar el servidor en la nube seleccionando el plan, proveedor de nube, región geográfica y nombre del clúster.

  • Crear un usuario de base de datos

    Definir un usuario y contraseña que permitirán acceder y administrar la base de datos.

  • Configurar el acceso a la red

    Autorizar las direcciones IP que podrán conectarse al clúster para garantizar la seguridad del acceso.

  • Crear la base de datos

    Acceder al clúster, abrir la opción de colecciones y crear una nueva base de datos junto con su primera colección.

  • Verificar la creación

    Comprobar que la base de datos y la colección aparecen correctamente en el panel de MongoDB Atlas.
